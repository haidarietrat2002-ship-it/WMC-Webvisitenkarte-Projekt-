# WMC-Webvisitenkarte-Projekt-
Willkommen auf meiner Webvisitenkarte
Projektbeschreibung
Diese Webvisitenkarte wurde im Rahmen des Unterrichts im Fach Web & Mobile Computing (WMC) erstellt.
Ziel des Projekts ist es, grundlegende Kenntnisse in HTML, CSS und JavaScript praktisch anzuwenden und eine eigene, interaktive Website zu gestalten.
Die Website stellt meine Hobbys, mein Interesse an Webentwicklung sowie mein aktuelles Projekt „Typ & Rush“ vor.
Hobby-Karten mit Flip-Animation (TOP)
Im oberen Bereich der Website befinden sich Hobby-Karten mit einer Flip-Animation.
Beim Anklicken einer Karte dreht sie sich und zeigt einen kurzen Text über das jeweilige Hobby.
Dieser Bereich dient dazu, einfache JavaScript-Interaktionen und CSS-Animationen umzusetzen und die Seite interaktiv zu gestalten.
Mini-Demo „Typ & Rush“ (BODY)
Im Hauptbereich der Website wird mein Spielprojekt „Typ & Rush“ vorgestellt.
Durch einen Klick auf den hervorgehobenen Namen Typ & Rush wird ein kurzes Demo-Video (ca. 30 Sekunden) abgespielt.
Das Video gibt einen kurzen Überblick über das Spielkonzept und die Funktionsweise.
Nach dem Abspielen schließt sich das Video automatisch und die Website wird wieder angezeigt.
Feedback-Bereich (FOOT)
Im unteren Bereich der Website befindet sich ein Feedback-Bereich.
Hier können Besucher:innen Feedback abgeben:
Allgemeines Feedback zur Website
Feedback zum Spiel Typ & Rush
Anmerkungen oder Verbesserungsvorschläge
Das Feedback wird über ein Formular eingegeben und nach dem Absenden erscheint eine Bestätigungsmeldung.

&ein Projekt auf GitHub zu dokumentieren und zu verwalten.
